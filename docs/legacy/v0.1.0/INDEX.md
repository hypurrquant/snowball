## 버전 규칙

```
v{MAJOR}.{MINOR}.{PATCH}

MAJOR: 구조 변경, 호환 불가 (컨트랙트 재배포 등)
MINOR: 기능 추가, 섹션 추가
PATCH: 오타, 주소 수정, 소규모 업데이트
```

## 상태 정의

| 상태 | 설명 |
|------|------|
| **Draft** | 초안. 리뷰/승인 전. 변경 가능성 높음 |
| **Review** | 리뷰 중. 피드백 반영 단계 |
| **Active** | 승인됨. 현재 운영 기준 문서 |
| **Archive** | 더 이상 유지보수하지 않음. 참고용 |
| **Deprecated** | 폐기. 새 문서로 대체됨 |
